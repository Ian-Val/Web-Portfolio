Python Implementation of the LZ77 Compression Algorithm

This project is the result of my final project within my Operating Systems class where I collaborated with Stephen Kussow on a presentation about the LZ77 compression algorithm.

The LZ77 algorithm is utilized within the Microsoft operating system to fit more pages within the RAM. Our Python implementation of the algorithm reduced file sizes by an average of 10%.

I worked on the algorithm implementation which consisted of compression, decompression, serialization, and deserialization of the input data. Kussow worked on the data collection and data visualization using Excel. We both collaborated on the presentation and the essay.
